# PrintableSacredGeometry
Scripts and STLs for expressing sacred geometry
- tools used OpenSCAD, PyMeshLab
Trimesh: This is the industry-standard pure Python library for loading, manipulating, and querying 3D meshes (STL, OBJ, etc.). It integrates well with 
Numpy and is widely used for algorithmically generating structures, checking if meshes are watertight (manifold), and exporting them for 3D printing
